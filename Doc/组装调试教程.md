# MMX-AR补充说明
1. 如果是拆下的鲍登头, 会比标准鲍登头大一圈, 要打印非标鲍登头版本的打印件
2. 996R的圆型舵机盘需要保留, 通过3个m2自攻螺丝与打印件结合, 更不容易打滑, 若皮带容易脱出, 确定长度之后可以上胶水
3. 凸轮轴固定使用M3机米螺丝, 通常4MM长够用, 若D轴较深请自己适当再加长, 若没有机米螺丝, 可以直接胶水或者其他方式固定
4. 舵机下压角度请确保完全垂直, 否则可能会打滑
5. 如果钢珠和微动中间有间隙, 可以垫合适厚度的胶带处理
